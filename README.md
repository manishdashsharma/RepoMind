## What is RepoMind?

RepoMind is an open-source CLI tool that lets you have natural conversations with any codebase вҖ” in **English or Hindi** вҖ” without sending a single byte to the cloud.

It indexes your code locally, stores vectors in a local Qdrant database, and uses Ollama to run language models on your own hardware. No API keys. No subscriptions. No privacy concerns.

```bash
$ repomind

 вҡЎ buddy is listening

  1.  Ask a question about your code
  2.  Index a new project

  You вҶ’ buddy: auth middleware kahan hai aur kaise kaam karta hai?

  рҹӨ– buddy:

  The auth middleware lives in `src/middleware/auth.js` (lines 12вҖ“48).
  It validates JWT tokens using the `jsonwebtoken` library...
```

---

## Features

- **100% local** вҖ” Ollama + Qdrant, zero cloud calls
- **Smart hardware detection** вҖ” suggests the right model for your machine (RAM, GPU, Apple Silicon)
- **Bilingual** вҖ” ask questions in English or Hindi, get answers in the same language
- **Conversation memory** вҖ” follow-up questions use previous context
- **Any language** вҖ” Python, JS, TS, Go, Rust, Java, C++, and 40+ more
- **Gitignore-aware** вҖ” skips `node_modules`, build artifacts, binary files
- **Safety guardrails** вҖ” blocks indexing of system directories, warns about `.env` files
- **Session logs** вҖ” every conversation is saved to `~/.repomind/sessions/`
- **Beautiful CLI** вҖ” Rich-powered colorful terminal with progress bars, spinners, tables
- **`repomind status`** вҖ” health check for all services at a glance

## Quick Start



## How It Works

```
Your Code
    в”Ӯ
    в–ј
  Indexer          Walk repo вҶ’ skip ignored dirs вҶ’ chunk files (400 tokens, 80 overlap)
    в”Ӯ
    в–ј
  Embedder         nomic-embed-text via Ollama вҶ’ 768-dim vectors
    в”Ӯ
    в–ј
  Qdrant           Store in collection repomind_{project} (cosine similarity)
    в”Ӯ
    в–ј  в”Җв”Җв”Җв”Җ at query time в”Җв”Җв”Җв”Җ
    в”Ӯ
  Embedder         Embed your question with the same model
    в”Ӯ
    в–ј
  Retriever        Top-8 most similar chunks from Qdrant
    в”Ӯ
    в–ј
  LLM              Build prompt with context вҶ’ stream answer via Ollama
    в”Ӯ
    в–ј
  Terminal         Streamed tokens, then source file citations
```

## Model Selection Guide

RepoMind auto-detects your hardware and picks the best model:

| Your Machine | Suggested Model | Quality |
|-------------|----------------|---------|
| < 8 GB RAM | `phi3.5:3.8b` | вҳ…вҳ…вҳҶвҳҶвҳҶ Lite |
| 8вҖ“16 GB RAM | `llama3.2:3b` | вҳ…вҳ…вҳ…вҳҶвҳҶ Good |
| 16+ GB RAM | `llama3.1:8b` | вҳ…вҳ…вҳ…вҳ…вҳҶ Great |
| Apple Silicon 16GB+ | `llama3.1:8b` | вҳ…вҳ…вҳ…вҳ…вҳҶ Great |
| Apple Silicon 32GB+ | `llama3.3:70b` | вҳ…вҳ…вҳ…вҳ…вҳ… Best |
| NVIDIA GPU вүҘ 8GB VRAM | `llama3.1:8b` | вҳ…вҳ…вҳ…вҳ…вҳҶ Great |
| NVIDIA GPU вүҘ 24GB VRAM | `llama3.3:70b` | вҳ…вҳ…вҳ…вҳ…вҳ… Best |

> **Hindi support:** `llama3.1:8b` and above handle Hindi well. Lighter models (3B and below) have limited Hindi understanding.

## Session Logs

After every chat session, RepoMind writes a structured markdown log to `~/.repomind/sessions/`:

```
~/.repomind/sessions/
  2024-01-15_14-32-07.md
  2024-01-16_09-45-22.md
```

Each log contains:
- Session summary (agent, duration, question count)
- List of projects queried and indexed
- Full Q&A transcript

## Privacy

RepoMind is designed from the ground up to keep your code private:

- **No cloud calls** вҖ” Ollama and Qdrant both run on `localhost`
- **No telemetry** вҖ” zero analytics, no crash reporting, nothing leaves your machine
- **No accounts** вҖ” no sign-up, no API keys
- **Vectors are local** вҖ” stored in a Docker volume on your machine
- **Config is local** вҖ” `~/.repomind/config.json`, readable only by you

## FAQ

**Q: How is this different from GitHub Copilot or Cursor?**  
A: Those send your code to external servers. RepoMind runs entirely on your machine вҖ” your code never leaves.

**Q: Does it work on Windows?**  
A: Not officially tested yet. Docker and Ollama support Windows, so it should work. PRs welcome!

**Q: Can I use a different LLM provider?**  
A: Currently Ollama-only. Support for other local providers is on the roadmap.

**Q: How do I update the model after install?**  
A: Run `repomind install` again and choose a different model when prompted.

**Q: My Hindi answers aren't great. What model should I use?**  
A: Use `llama3.1:8b` or larger. Smaller models have limited multilingual support.
